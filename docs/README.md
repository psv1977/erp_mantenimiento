# Confiabilidad Industrial (RCM App)

Sistema de gestión para el **Mantenimiento Centrado en Confiabilidad (RCM)** diseñado para transformar 
la toma de decisiones en entornos industriales, eliminando la dependencia de hojas de cálculo vulnerables.

## 🚀 Propuesta de Valor
A diferencia de las planillas tradicionales, esta aplicación ofrece:
- **Integridad de Datos:** Reglas de validación que impiden errores de cálculo en el AMEF.
- **Accesibilidad:** Interfaz web intuitiva para uso en planta y oficina.
- **Automatización:** Generación de matrices de criticidad y planes de mantenimiento con un clic.

## 🛠️ Roadmap del Proyecto
1. [ ] Configuración del entorno y estructura de datos inicial.
2. [ ] Desarrollo del módulo de **Análisis de Criticidad**.
3. [ ] Implementación de lógica de **Modos y Efectos de Falla (AMEF)**.
4. [ ] Dashboard de indicadores (Disponibilidad, Confiabilidad $R(t)$).
5. [ ] Generación automática de reportes en PDF.

## 💻 Stack
- **Backend/Frontend:** Python + Streamlit.
- **Análisis:** Pandas / NumPy.
- **Control de Versiones:** Git & GitHub.
